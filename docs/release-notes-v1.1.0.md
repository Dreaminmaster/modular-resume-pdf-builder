# v1.1.0 Product Polish Release

## Positioning

**中文：一份信息池，多份定制简历。**  
**English: One information pool. Multiple tailored resumes.**

## Release focus

This release is about product polish rather than feature expansion.

Key improvements include:
- better mobile focus on the main workflow
- clearer editing hierarchy
- cleaner UI wording
- stronger visual consistency
- improved demo readiness
- refined PDF export behavior

## Highlight updates

### Product-facing README
The README has been rewritten as a product landing page instead of a technical document.

### Demo datasets
Four official demo datasets are included:
- Engineering Student
- Product Manager
- Study Abroad Application
- English Resume

### Mobile product feel
The mobile experience now focuses more clearly on:
- editing
- preview
- export

### Content scope workflow
Unchecked modules no longer appear in the editor, but their content remains preserved.

### Visual polish
The UI now has stronger hierarchy, better readability, and more product-like panel focus.

## Demo files

- `demo-engineering-student.json`
- `demo-product-manager.json`
- `demo-study-abroad.json`
- `demo-english-resume.json`

## Notes

This release does **not** aim to turn the project into a rich text editor or a canvas design tool.

The product direction remains focused on:
- structured information pool
- module-based tailoring
- fast role-specific resume generation
- clean PDF export
